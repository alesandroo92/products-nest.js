## Description

Api Rest basada en Nest.js de productos guardados en la base de datos MongoDB.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
